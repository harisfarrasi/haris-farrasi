# **App Name**: Haris Farrasi

## Core Features:

- Mobile-First Design: Website is designed with a mobile-first approach for optimal viewing on smaller devices.
- Scroll Narrative Content: Presents content in a scrollable narrative format to provide a seamless user experience when reading about missions and projects.
- Headless CMS Integration: Serves project info, writings and other meta content from Sanity.io through static APIs for easy updates without full rebuilds.
- Privacy-Friendly Analytics: Utilizes Plausible Analytics for tracking, optimized to be privacy-friendly without third-party bloat.

## Style Guidelines:

- Primary color: Steel blue (#4682B4) for a modern, futuristic, and high-contrast feel.
- Background color: Dark charcoal (#1A1A1A), a desaturated version of the primary hue, creating contrast while remaining easy on the eyes.
- Accent color: Cerulean (#1E90FF), analogous to steel blue but brighter and more saturated, to draw attention to key interactive elements.
- Headline font: 'Sora' (sans-serif) for headings, offering a modern and readable style.
- Body font: 'Inter' (sans-serif) for body text, ensuring readability and clean presentation.
- Implements a flex-grid hybrid layout to ensure sections maintain semantic layer separation and harmonize across different screen ratios.
- Uses smooth, subtle animations with Framer Motion to improve narrative flow.
